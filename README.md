# Uber Ride Analysis Dashboard

This project contains a Power BI dashboard for analyzing Uber ride performance across different vehicle categories. The report focuses on booking value, trip distance, driver ratings, and total distance travelled to help compare service types such as Auto, Go Mini, Go Sedan, Bike, Premier Sedan, eBike, and Uber XL.



## Dashboard Overview

The report provides a vehicle-type level summary of Uber ride activity. From the dashboard, you can quickly compare:

- Total booking value by vehicle type
- Average ride distance
- Sum of driver ratings
- Average or total distance travelled

The visual shown in the report highlights which vehicle categories contribute the most to revenue and ride activity.

## Key Insights From the Shown Report

- `Auto` has the highest booking value in the displayed table
- `Go Mini` and `Go Sedan` are also strong contributors to overall bookings
- Average ride distance stays fairly consistent across most vehicle types at around `24 km`
- `Auto` and `Go Mini` lead in driver rating totals, suggesting high ride volume
- `Uber XL` has the lowest contribution among the visible categories in this view
